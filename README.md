# Movie Recommendation System

## Description
A C++ mini project based on Object-Oriented Programming and Sorting Algorithms.

## Features
- Add Movies
- Rate Movies
- Display Movies
- Sort by Rating
- Sort by Year
- Sort by Title Length

## Concepts Used
- Inheritance
- Polymorphism
- Abstraction
- Vectors
- Selection Sort
- Insertion Sort
- Merge Sort

## Language Used
C++

## IDE Used
Dev C++
